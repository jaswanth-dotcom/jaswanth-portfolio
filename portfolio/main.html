About Me

Projects

Easy Travel App

Easy Travel is an all-in-one travel companion app designed in Figma, offering location tracking, smart destination suggestions, expense management, and hotel booking features. The user interface focuses on seamless navigation and a modern, intuitive design to enhance the travel experience.

Home Service App

Home Service Application is a user-centric app designed in Figma to simplify access to essential services like plumbing, cleaning, and electrical repairs. The interface is clean, intuitive, and optimized for fast bookings, ensuring a seamless experience for users and service providers alike.

Movie Review Site

Movie Review Site is a responsive UI project designed in Figma for discovering and reviewing movies and series. It features trending titles, user ratings, and top picks, all presented through a clean, modern layout focused on usability and visual appeal.
